# react-notes-app

# frontend: react
# backend: postgres, node/express

## features

1. login/signup authentication, with jwt tokens and hashed passwords
2. postgres database to store tasks
3.  create, update, search for tasks
4.  track the progress of the tasks
5.  responsive
